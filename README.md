# hikkaCLI
CLI tool to interact with hikka.io

### **Features**:
- [x] Translate characters from anime
- [ ] Bulk translate
- [x] Find words in description of chraracter via edits

## Build
1. Clone repo

```bash
git clone https://github.com/rosset-nocpes/hikka_cli
```

3. Build with cargo:

```bash
cargo build
```

Or just run:
```bash
cargo run
```
